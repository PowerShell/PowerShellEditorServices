Type Name Description
System.String name

The display name for the host, typically in the form of "[Application Name] Host".
System.String profileId

The identifier of the PowerShell host to use for its profile path. loaded. Used to resolve a profile path of the form 'X_profile.ps1' where 'X' represents the value of hostProfileId. If null, a default will be used.
System.Version version

The host application's version.
